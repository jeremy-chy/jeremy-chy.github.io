# Academic Personal Website

A modern, minimalist academic personal website template.

## Features

- Clean and sophisticated design
- Responsive layout for all devices
- Smooth animations and interactive elements
- Easy to customize and maintain

## Setup Instructions

1. **Add Your Profile Photo**: Place your profile photo in the root directory and name it `profile.jpg` (or update the `src` attribute in `index.html` if using a different name/format). Recommended size: 280x280 pixels or any square image.

2. **Customize Content**: Edit `index.html` to update:
   - Your name and title
   - Affiliation and contact information
   - Bio/introduction
   - Research interests (currently 3 placeholder cards)
   - Publications list

3. **Deploy**: Since this is a GitHub Pages repository, simply push your changes to the `main` branch and your site will be live at `https://jeremy-chy.github.io`.

## File Structure

```
├── index.html      # Main HTML file
├── styles.css      # All styling and animations
├── script.js       # Interactive JavaScript features
└── README.md       # This file
```

## Customization

- **Colors**: Modify CSS variables in `styles.css` (defined in `:root`)
- **Fonts**: Currently using Google Fonts (Inter). Change the font import in `index.html` if desired
- **Sections**: Add or remove sections by following the existing structure

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).
